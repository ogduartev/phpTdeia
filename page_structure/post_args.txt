actionLevel             : Profundidad de Matriz en las columnas de acciones
factorLevel             : Profundidad de Matriz en las filas de factores
matrixType              : Tipo de contenido a desplegar en la matriz e indicador para calcularlo, según el formato
                                 tipo-id   
                                      tipo puede ser: 'aggregations','effects','propierties'
                                      
                                      aggregations: id es el indicador del agregador
                                      effects: si id <0 es el núnmero de efectos, en caso contrario es el id del agregador de importancia
                                      propierties: id es el id de la propiedad
cellType                : Tipo de contenido de la celda de la matriz. Puede ser  'Short', 'Number', 'Number/Ambiguity', 'Color'
varId                   : Id de la variable a ser desplegada. Este id puede corresponder a valriables de effect_propierties o de importances
                             para el número de efectos es factorId-actionId
